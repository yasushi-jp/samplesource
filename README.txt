このZIPファイルは、書籍「Java EE 7 徹底入門」のサンプルプログラムとサンプルデータです。

○フォルダ構成
・サンプルソース
　- knowledgebank ・・・ ナレッジバンクの実装
　- knowledgebank_client ・・・ JAX-RSクライアントのサンプル
　- SimpleBatch  ・・・ ナレッジバンクに含まないバッチ実装サンプル

・サンプルデータ
　- data4jBatch.sql ・・・ jBatch(第10章)の説明で利用するデータを準備する SQL 文
　- data4JPA.sql ・・・ JPA(第7〜8章)の説明で利用するデータを準備する SQL 文
　- data4JSF.sql ・・・ JSF(第1〜3章)の説明で利用するデータを準備する SQL 文

・ナレッジバンク動作ガイド
　- ナレッジバンク動作ガイド.html ・・・ ナレッジバンクを動作させるための手順書

○サンプルの動作確認方法
「ナレッジバンク動作ガイド」フォルダにある「ナレッジバンク動作ガイド.html」を参照し設定を行って下さい。
サンプルデータの登録方法については、上記 html の「３−２．サンプルデータの登録」の項を参照してください。
